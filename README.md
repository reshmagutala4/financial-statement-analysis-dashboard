Developed an interactive dashboard analyzing 5-year financial performance.

- Microsoft Excel (Pivot Table, Financial Ratios)
- Power BI
- Financial Statement Analysis

- Profitability Ratios
- Efficiency Ratios
- Trend Analysis

- Evaluated working capital efficiency
- Assessed financial stability
